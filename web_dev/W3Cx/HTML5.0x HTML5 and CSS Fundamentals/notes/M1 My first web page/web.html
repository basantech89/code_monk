1.1 The Web

Sir Tim Berners-Lee wrote the proposal in 1989 for www, then created first web browser, server & web page, wrote first specification for URLs, HTTP and HTML Lee Founded W3C in Oct 1994 at MIT in collab with CERN April 1995, Inria became the first European W3C host The Web is an application of the Internet. The Web is the most popular way of accessing the Internet, but other applications of the Internet are e-mail and ftp for example.
