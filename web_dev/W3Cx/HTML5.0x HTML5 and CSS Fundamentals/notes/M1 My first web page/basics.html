Some Examples

<ol start="5"> - Start an ordered list with number 5 instead of 1

Usually attributes are name value pairs, only exception is if the attribute is a boolean attribute, instead of writing true or false, we add the attribute name to indicate true and omit it to indicate false. As an example is the reversed attribute in an ordered list <ol> causing the list to be reversed. <ol start=5 reversed> would result in 5, 4, 3

id - style a specific element
class - style a group of elements
