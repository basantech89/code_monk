5.1.2 The thead, tbody and tfoot tags

It is very useful to define parts of a table as header, body and footer because once browsers are able to identify which cells are header and footer, the body can be allowed to scroll independently of header and footer catering to a good table viewing experience in small screens

thead

  • Just like how we use <colgroup> to group columns, <thead> is used to group the header content in a HTML table.
  • As we learned in the previous unit, header cells are specified using <th> as a child of <tr>. Rows specified within <thead> indicate that they are header rows.

tbody

Following <thead>, subsequent rows are considered body rows in a table. Regular cells are specified using <td> as a child of <tr>
